# PDF-to-PNG

スマホのブラウザ上だけでPDFをPNG画像に変換する、X投稿向けのシンプルなPWAです。PDFファイルは外部サーバーへアップロードされず、PDF.jsでブラウザ内処理します。

## 機能

- PDFファイルを選択して読み込み
- PDFの各ページを白背景PNGへ変換
- 変換倍率を `1x` / `2x` / `3x` から選択
- X投稿向けに横幅を `1600px` / `2000px` へリサイズ、または元サイズで出力
- 変換後のページ別プレビュー表示
- `元ファイル名_page-01.png` 形式で個別ダウンロード
- 全ページをZIPで一括ダウンロード
- 4ページごとに「X投稿セット1」「X投稿セット2」形式で表示
- PWA対応（ホーム画面に追加可能）

## 使い方

1. `index.html` をブラウザで開きます。
   - PWAとして安定して使う場合は、ローカル/静的Webサーバーで配信してください。
   - 例: `python3 -m http.server 8000` を実行し、`http://localhost:8000/` を開きます。
2. 「PDFファイルを選択」からPDFを選びます。
3. 変換倍率を `1x` / `2x` / `3x` から選びます。
4. 必要に応じてX投稿向け横幅を `1600px` または `2000px` にします。
5. 「PNGに変換」を押します。
6. プレビューを確認し、各ページの「このページを保存」または「全ページをZIPで保存」から保存します。

## Android Chromeでホーム画面に追加する

1. HTTPSまたはローカルホストなど、PWAを利用できるURLでページを開きます。
2. Chromeのメニューから「ホーム画面に追加」を選択します。
3. 追加後はホーム画面のアイコンから起動できます。

## 注意事項

- PDF.jsとJSZipはCDNから読み込んでいます。初回利用時やキャッシュがない場合はネットワーク接続が必要です。
- 大きなPDFや高倍率変換では、スマホのメモリ使用量が増えます。処理が重い場合は倍率を下げるか、ページ数の少ないPDFでお試しください。
- すべての変換処理はブラウザ内で実行されます。選択したPDFの内容はこのアプリから外部サーバーへ送信しません。
